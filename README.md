# TEJO Theme
The **TEJO** theme for [Grav CMS](http://github.com/getgrav/grav). 

## Support Targets
Browserslist `>= 2%`:

- Chrome 77
- Firefox 70
- Safari 12.2
- UC Browser 12.12
- Samsung Internet 10.1

## Installation
Run `npm run build` upon installation to compile the CSS.
Also run `composer install` to install php dependencies.
